# ViewPCLens

P.C.Lens のコメントを取得する機能、コメント投稿サイトを iframe で表示する機能を実装しています。  
スクロール量と URL をコピペして使用してください。

This Chrome extension get comment for P.C.Lens.
One more thing, it get scroll level and URL from active tabs.
